Accend Link Vanilla JS Demo

This example demonstrates how to integrate the Accend Link SDK with vanilla JavaScript. The SDK is automatically loaded and initialized when the page loads.

Note: This demo automatically initializes AccendLink with a demo customer access token. Replace with your actual token in a production environment.

Implementation Example

Auto-Initialize Implementation:
// 1. Define SDK URLs
const LINK_SDK_URL = "https://static.withaccend.com/sdk/v1/link.min.js";
const LINK_SDK_STAGING_URL = "https://static.withaccend.com/sdk/staging/link.min.js";

// 2. Auto-initialize on page load
document.addEventListener('DOMContentLoaded', async () => {
  const container = document.getElementById('accend-link-container');
  
  try {
    // Load SDK
    const module = await import(LINK_SDK_STAGING_URL);
    
    // Initialize immediately
    const instance = new module.AccendLink({
      target: container,
      props: {
        config: {
          data: {
            customerAccessToken: "your-customer-access-token"
          },
          component: {
            theme: {
              primary: "#2196f3"
            }
          }
        },
        onSuccess: () => console.log("Success!"),
        onFail: (error) => console.error("Error:", error)
      }
    });
  } catch (error) {
    console.error("Failed to load AccendLink:", error);
  }
});
Configuration Options:
const config = {
  data: {
    customerAccessToken: "required-customer-token" // Generated server-side
  },
  component: {
    theme: {
      primary: "#your-brand-color"  // Optional theme customization
    }
  }
};
