Accend Link Vanilla JS Demo

This example demonstrates how to integrate the Accend Link SDK with vanilla JavaScript. The SDK is loaded dynamically from the CDN and initialized with a target DOM element.

Note: Make sure to replace the demo customer access token with your actual token in a production environment.

Implementation Example

Basic Implementation:
// 1. Define SDK URLs
const LINK_SDK_URL = "https://static.withaccend.com/sdk/v1/link.min.js";
const LINK_SDK_STAGING_URL = "https://static.withaccend.com/sdk/staging/link.min.js";

// 2. Get container element
const container = document.getElementById('accend-link-container');

// 3. Load and initialize SDK
const module = await import(LINK_SDK_STAGING_URL);
const instance = new module.AccendLink({
  target: container,
  props: {
    config: {
      data: {
        customerAccessToken: "your-customer-access-token"
      },
      component: {
        theme: {
          primary: "#2196f3"
        }
      }
    },
    onSuccess: () => console.log("Success!"),
    onFail: (error) => console.error("Error:", error)
  }
});

// 4. Control the SDK
instance.open();  // Open the interface
instance.close(); // Close the interface
Configuration Options:
const config = {
  data: {
